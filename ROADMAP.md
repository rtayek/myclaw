# MyClaw Project Roadmap

## Phase 1: Core Workbench & Backends (Current)
- [x] Local 3-tier desktop runtime (Java UI, Core Broker, Backend Engine).
- [x] Socket transport integration (`127.0.0.1` TCP loopback).
- [x] Local Ollama (`glm4:9b`) and Claude CLI backend support.
- [x] Dual append-only transcript logging (`.jsonl` telemetry and `.md` transcripts).
- [x] High-contrast accessible Swing UI with screen reader compatibility.

## Phase 2: Enhanced Accessibility & Interaction (In Progress)
- [ ] Hands-free speech-to-text (STT) dictation and text-to-speech (TTS) output.
- [ ] Tonal audio sonification for build, test, and background agent state events.
- [ ] Improved spatial window management and global hotkey bindings (`Win+V`).

## Phase 3: Knowledge Base & Multi-Agent Collaboration (Future)
- [ ] Automatic extraction of durable personal knowledge from chat histories.
- [ ] Multi-model comparison and advisor harness (escalating complex tasks to cloud frontier models).
- [ ] Advanced 24/7 background agent automation and scheduled task loops.
