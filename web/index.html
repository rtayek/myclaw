The post-frontier harness architecture separates context ownership from raw inference compute:

  • Local Deskside Orchestration: The primary agent loop runs on local hardware (Ollama / unified memory). Everyday queries, tool invocation, and private context remain 100% on your machine at zero token cost.
  • Remote Advisor Escalation: Frontier cloud models are invoked strictly as remote sub-agents only when deep reasoning is required, returning structured results back to the local harness.
  • Data Sovereignty: Conversations are captured at transaction time in standard Markdown/JSON formats locally. Switching backends requires no cloud vendor permission—simply rehydrating your local context records.
